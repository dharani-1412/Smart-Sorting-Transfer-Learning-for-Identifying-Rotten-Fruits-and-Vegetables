NutriGaze Analysis Details

This project uses transfer learning with the VGG16 model to classify fruits and vegetables into 28 categories of fresh and rotten.

  • Model: VGG16
  • Input Image Size: 224x224
  • Training Dataset: 28 classes (Fresh/Rotten)
  • Frameworks Used: Flask, TensorFlow, Keras
  • Prediction Output: Label of class (e.g., "Apple - Rotten")
